# Simple Islamic Calendar — Public Documents

This repository hosts the **public-facing documents** for the
**Simple Islamic Calendar** mobile app. It exists so these documents have
stable, publicly accessible URLs that can be referenced from the app itself and
from the App Store and Google Play store listings.

It contains no application source code — only documentation.
